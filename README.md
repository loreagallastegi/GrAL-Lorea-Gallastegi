# GrAL-Lorea-Gallastegi
Tropikoz kanpoko zikloien sakontzearen analisia egiteko garatutako Python-eko kodea


## Deskribapena

Errepositorio honetan Fisikako graduko GrAL-a egiteko garatutako Pythonen egindako kodea dago.

Helburua tropikoz kanpoko zikloien sakontzea aztertzea da, presioaren aldaketa eragiten duten mekanismo fisiko desberdinak aztertuz.

## Aztertutako kasuak

Honako zikloiak aztertu dira:

- Xynthia (2010)
- Klaus (2009)
- Braer (1993)
- Fastnet (1979)


## Exekuzioa

Kode nagusia exekutatzeko:

python zikloiak_deitu.py
